# SRT-Osint
Um Osint certificado que usa a db do sherlock 
e mas alguns sites em seu banco local de 
sites. Nesse osint tem um limite de usos de 3 
usos quando você usa 3 vezes você vai ter que 
esperar 4 segundos para ultilizar denovo.
# Informações
O SRT Osint ele é um Osint de terminal como
muitos chamam ele roda em: termux ou qualquer 
outro terminal e em aplicativos de python 
como pydroid.
# Instalação
Para instalar o SRT Osint você deve usar:
git clone https://github.com/NO-RH/Srt-Osint

cd Srt-Osint
python3 SRT.py
